.. Title

cisco.ise.self_registered_portal -- Resource module for Self Registered Portal
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.ise collection <https://galaxy.ansible.com/cisco/ise>`_ (version 0.0.9).

    To install it use: :code:`ansible-galaxy collection install cisco.ise`.

    To use it in a playbook, specify: :code:`cisco.ise.self_registered_portal`.

.. version_added

.. versionadded:: 1.0.0 of cisco.ise

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Manage operations create, update and delete of the resource Self Registered Portal.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.
