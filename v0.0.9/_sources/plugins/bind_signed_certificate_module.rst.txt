.. Title

cisco.ise.bind_signed_certificate -- Resource module for Bind Signed Certificate
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.ise collection <https://galaxy.ansible.com/cisco/ise>`_ (version 0.0.9).

    To install it use: :code:`ansible-galaxy collection install cisco.ise`.

    To use it in a playbook, specify: :code:`cisco.ise.bind_signed_certificate`.

.. version_added

.. versionadded:: 1.0.0 of cisco.ise

   :depth: 1

Synopsis
--------

.. Description

- Manage operation create of the resource Bind Signed Certificate.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.
