cisco.ise.active_directory_add_groups – Resource module for Active Directory Add Groups

Note

This plugin is part of the cisco.ise collection (version 1.2.1).

To install it use: ansible-galaxy collection install cisco.ise.

To use it in a playbook, specify: cisco.ise.active_directory_add_groups.

New in version 1.0.0: of cisco.ise

Synopsis

  • Manage operation update of the resource Active Directory Add Groups.

Note
