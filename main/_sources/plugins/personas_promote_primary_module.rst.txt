.. Title

cisco.ise.personas_promote_primary module -- Promote a node as the primary node
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `cisco.ise collection <https://galaxy.ansible.com/cisco/ise>`_ (version 2.5.16).

    To install it, use: :code:`ansible-galaxy collection install cisco.ise`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.cisco.ise.personas_promote_primary_module_requirements>` for details.

    To use it in a playbook, specify: :code:`cisco.ise.personas_promote_primary`.
